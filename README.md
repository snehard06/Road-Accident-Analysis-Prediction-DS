# 🚦 Road Accident Analysis & Prediction (India)

## 📌 Project Overview

This project analyzes road accident data in India using Data Science techniques to uncover patterns, identify key risk factors, and generate insights for improving road safety. It demonstrates a complete workflow including data cleaning, visualization, and insight generation to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze accident trends across states and time
* Identify factors affecting accident severity
* Study impact of speed, weather, and driver demographics
* Generate actionable safety insights
* Build a foundation for predictive analysis

---

## 📊 Dataset Features

The dataset includes key attributes such as:
State & City, Year & Time, Accident Severity, Vehicles Involved, Casualties & Fatalities, Weather Conditions, Road Type & Condition, Lighting, Speed Limit, Driver Age & Gender, and Alcohol Involvement.

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Jupyter Notebook

---

## 🔧 Project Workflow

**1. Data Understanding:** Explored dataset structure and variables
**2. Data Wrangling:** Handled missing values, removed duplicates, cleaned data
**3. Visualization:** Used bar plots, pie charts, histograms, scatter plots, heatmaps, and box plots
**4. Insights:** Identified patterns in accidents, severity, and risk factors
**5. Recommendations:** Suggested improvements for road safety

---

## 📈 Key Insights

* Drivers aged 20–40 are most accident-prone
* Higher speeds increase casualties and severity
* Weather conditions like rain and fog raise accident risk
* Night-time accidents are more severe
* Certain states show consistently higher accident rates
